# stm32n6570-dk

### Set up compiler
    $ sudo apt install gcc-arm-none-eabi

### Get sources and build the project
    $ git clone --recursive https://github.com/mdepx/stm32n6570-dk
    $ cd stm32n6570-dk
    $ make clean all
